# titanic-survival-prediction

## Project Overview
This project predicts passenger survival in the Titanic disaster using machine learning.

## Dataset
Dataset from Kaggle Titanic Competition.

## Technologies Used
Python
Pandas
Scikit-learn
Matplotlib
Seaborn

## Workflow
1. Data cleaning
2. Exploratory data analysis
3. Feature engineering
4. Model training
5. Model evaluation



## How to Run
pip install -r requirements.txt
python src/train_model.py
